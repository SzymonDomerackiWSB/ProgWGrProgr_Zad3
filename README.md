# Snake

## Instrukcja

Aby sterować wężem(złożonym z symboli "O") należy używać strzałek.\
Wąż porusza się w danym kierunku bez możliwości zatrzymania się.\
Aby zwiększyć swój wynik, należy zbierać jedzenie, znaczone symbolem "X"\
W przypadku kolizji węża ze ścianą lub samym sobą, gra kończy się, wyświetlając wynik końcowy
